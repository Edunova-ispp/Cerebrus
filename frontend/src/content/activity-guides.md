## general

### maestro
Usa esta guía como base para cualquier actividad.

1) Pulsa Crear.
2) Elige tipo de actividad.
3) Rellena título, descripción y configuración.
4) Revisa que las instrucciones se entiendan.
5) Guarda y publica.

Puedes crear la actividad usando un modelo de IA. Como consejo: sé claro en tu petición, especifica qué tipo de actividad quieres crear y revisa el contenido tras la creación.

### alumno
Vamos paso a paso para que te salga genial.

1) Lee lo que te pide el juego.
2) Haz la actividad con calma.
3) Mira tu respuesta una vez más.
4) Pulsa enviar o terminar.

Consejo rápido:
- Si tienes dudas, vuelve a leer la consigna.

3) Añade la URL de una imagen si aporta valor.

## tablero

### maestro
Esta actividad se basa en que el alumno debe recorrer un tablero respondiendo a preguntas correctas. Los alumnos pueden, tras cada pregunta, elegir el camino a seguir.

1) Define casillas o bloques del tablero.
2) Crea preguntas breves por casilla.
3) Decide si se le muestra al alumno la respuesta tras fallar o no, para que avancen aunque fallen.
4) Guarda para crearla.

Puedes crear la actividad usando un modelo de IA. Como consejo: sé claro en tu petición, especifica qué tipo de actividad quieres crear y revisa el contenido tras la creación.

### alumno

2) Lee la pregunta y responde.
3) Sigue hasta completar las preguntas necesarias.

## carta

### maestro

En esta actividad los alumnos deben, de dos en dos, levantar las cartas para emparejarlas, su pregunta con su respuesta. Si fallan, se voltean de nuevo.

1) Define, para cada par de cartas, una pregunta y una respuesta.
2) Redacta contenido breve y claro en cada carta.
3) Comprueba que las relaciones tengan sentido.
4) Guarda para crearla.

Puedes crear la actividad usando un modelo de IA. Como consejo: sé claro en tu petición, especifica qué tipo de actividad quieres crear y revisa el contenido tras la creación.

### alumno
1) Pulsa en las cartas para desvelarlas.
2) Recuerda las preguntas y las respuestas.
3) Completa todas las combinaciones.

1) Redacta pistas claras y su respuesta.
2) Guarda para crearla.

Puedes crear la actividad usando un modelo de IA. Como consejo: sé claro en tu petición, especifica qué tipo de actividad quieres crear y revisa el contenido tras la creación.

### alumno

2) Pulsa en las pistas o en la palabra para empezar a escribir.
3) Escribe la respuesta que creas correcta.
4) Usa las letras que se cruzan para ayudarte.
5) Pulsa en "Comprobar" para completarlo.


## pregunta-abierta

### maestro
Pregunta abierta de teoría. Se corrigen automáticamente usando nuestro modelo de IA, comparando la respuesta del alumno con la suya.

1) Escribe una pregunta concreta.
2) Evita enunciados demasiado amplios, el modelo corrector puede confundirse.
3) Guarda para crearla.

Puedes crear la actividad usando un modelo de IA. Como consejo: sé claro en tu petición, especifica qué tipo de actividad quieres crear y revisa el contenido tras la creación.

### alumno
1) Lee bien la pregunta.
2) Escribe tu respuesta con tus palabras.
3) Revisa que se entienda bien.
4) Pulsa enviar para terminar.
